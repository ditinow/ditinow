《一》云服务器选购及安装

范例以腾讯云为例，其它云服务的安装方法类似。


CC14区块链全节点硬件要求较低，1核1G既能满足目前需求。建议选择1核2G的3年促销款。


请选择Ubuntu 18.01.1 LTS。更新版本的Ubuntu也适用。


支付成功之后点击页面右上方的“控制台”进行下一步操作。


根据刚刚订购的服务器类型进入管理页面。


请选择“更多”->“关机”。之后我们需要将实例（云服务器）与密钥绑定。


请选择“密钥”->“创建密钥”。


请输入密钥名称，点击“确认”。刚刚创建的密钥将会被下载至默认的下载文件夹。


点击右方的“操作”下的“绑定/解绑实例”。


此时您应该只有一个密钥和实例可以绑定，✔实例并点击“确认”。


密钥绑定结束后请选择“服务器”->“更多”->“管理”。下一步将设置防火墙。


请选择“防火墙”->“添加规则”。我们需要为即将安装的CC14区块链全节点开放端口以便与其它节点交换数据。


端口请填写“19884-19886”，备注请填“CC14”；点击“确定”。


回到“服务器”->“概要”，选择开机。请记录页面左下方信息“实例IP”->“(公)xxx.xxx.xxx.xxx”

此时可以离开腾讯云控制台，之后的操作将通过远程登录在命令行下完成。

《二》配置MobaXterm

根据个人喜好选择Xterm终端。这里用MobaXterm举例。


点击MobaXterm左上方的“Session”设置登录刚刚创建并开机的腾讯实例（云服务器）。


点击对话框左上方的“SSH”，输入之前获得的实例IP（公）-公网IP，每段数字之间由小数点“.”间隔。✔Specify username 并输入“ubuntu”，这是腾讯云Ubuntu的默认用户名。
✔Use private key，点击淡蓝色文件图标，并选择您在上一步创建密钥时自动下载的文件。点击“OK”结束此次配置。

操作无误的前提下，您此时进入腾讯云Ubuntu实例的远程控制端口。我们将通过命令行完成CC14公链的安装及启动。

《三》安装及启动CC14区块链全节点

以下操作参考 DITINOW的gitee仓库文件Readme_CC14.txt

在终端分布输入以下命令行。可以依次将以下命令“拷贝/复制”到MobaXterm界面上。
sudo apt update
sudo apt upgrade -y
sudo apt install zip -y
sudo apt install dos2unix -y
sudo apt install openjdk-11-jre-headless -y
sudo apt install openjdk-11-jdk-headless -y
git clone https://gitee.com/ditinow/CC14.git

安装过程截图

需要的系统更新、运行环境及节点源码已经下载完成，下一步进行编译，系统防火墙配置及启动区块链节点。
cd cc14
sh ./compile.sh
sudo ufw allow 19884:19886/tcp
chmod 751 run.sh
chmod 751 stop.sh
./run.sh --daemon

系统防火墙配置有别于腾讯实例（云服务器）防火墙设置。节点将在后台运行，您可以关闭终端，这不会影响节点的正常运行。

您可以通过浏览器和实例的公共IP访问这个节点的在线钱包。请在浏览器输入以下链接，用实例的公网地址替代“xxx.xxx.xxx.xxx”
http://xxx.xxx.xxx.xxx:19886/

《四》区块链本地节点设置

以下使用本地工作站，这里以视窗工作站为例。

从gitee下载CC14.zip文件并解压。


请修改“cc14/conf/文件夹内的“nxt.properties”文件，将其中的“#”号去掉并保存：
nxt.adminPassword=
nxt.maxPrunableLifetime=7776000
nxt.maxUploadFileSize=10240000

“7776000”秒，即90天。这是可修剪的数据在工作站保存的时间。例如两个CC14钱包之间可以相互短信，为了保障工作站区块链数据包不至于过于臃肿，这类数据将在90天后从工作站自动删除。但用户仍可以通过云端全节点读取这些数据。

默认区块链存证文件上限为“512000”，即512。修改为“10240000”，10MB，能满足一般文本文件的存证需要，可根据您的需求进一步调整。

“nxt.properties”文件修改完成之后，如果您视窗环境的JAVA安装无误，您可以运行解压后文件夹内的“run.bat”启动CC14节点。

本地工作站运行节点的目的是方便用户在使用CryptoC14提供的DApp应用时，能够更快捷的使用本地的接口服务以及突破512KB的存证上限。请勿关闭这个CMD视窗。

局域网内的节点为公网提供API服务可以使用VPN或者通过路由器传递。
